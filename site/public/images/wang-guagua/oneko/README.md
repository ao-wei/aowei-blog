# Wang Guagua oneko assets

This directory contains the fixed `oneko.js` sprite for Wang Guagua.

- `wang-guagua-oneko-sprite.png`: final `256 x 128` RGBA sprite sheet used by the site.
- `frames/`: the 32 fixed `32 x 32` action frames extracted from the final sprite sheet.
- `frames/manifest.json`: frame names, row/column positions, and pixel coordinates in the sprite sheet.

The sprite follows the upstream `oneko.js` frame order: 8 columns by 4 rows, one `32 x 32` frame per cell.
